## Wiring steps
1. Set the `song-plotter` agent entry in `~/.openclaw/openclaw.json` to this workspace (`.../workspace-inkkeeper/agents/song-plotter`).
2. Restart the OpenClaw gateway.
3. Clear existing song-plotter sessions so the next Discord exchange in <#1483562668316037181> boots with the new persona.
4. Smoke-test (“which agent are you?”).

Add ops notes (gear, DAW references, playlists) to `TOOLS.md` once the lane stabilizes.
