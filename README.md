# Writer's Lantern

Writer's Lantern is a one-screen static app for writers who need a gentle daily push back into their draft.

## What it includes

- A rotating daily motivation message written for fiction writers.
- A mood selector (`stuck`, `tired`, `ready`, `anxious`) that adapts the motivation line.
- A `Generate another` button for cycling to a fresh prompt inside the selected mood.
- A tiny actionable prompt that lowers the cost of starting.
- A book-planning checklist with general story questions plus genre-specific sections.
- A one-click daily check-in with a lightweight streak.
- Local persistence through `localStorage`, so the daily state survives refreshes and browser restarts.

## Why these features

The app stays intentionally narrow:

- Small goals support motivation better than vague pressure.
- Consistent return cues matter more than waiting for a high-energy session.
- A visible streak and a concrete next step reduce friction and reward small wins.

## How to use it

Open [`index.html`](/Users/al/.openclaw/workspace-inkkeeper/index.html) directly in a browser, or serve the folder with any tiny static server.
